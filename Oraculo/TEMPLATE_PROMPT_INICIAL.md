# TEMPLATE Prompt Inicial — KISO WEB

## Contexto base
Proyecto KISO WEB — ecosistema digital del universo KISOLAND.
Stack: Next.js 15 + TS + Tailwind v3.
Theming modular: `app/brand.css` (tokens `--lam1-*` / `.theme-*`).
Estructura: `/app/`, `/components/ui/`, `/public/`.

## Filosofía
Apple-like (claridad, ritmo, espacio) + narrativa simbólica.
Héroes con degradé solo en franja.  
Cards con `--surface` crema 94% + shadow limpia.  
Tokens y paletas por lámina (PDF “Tipografías y Color”).

## Formato del prompt inicial
- **Objetivo del hilo**  
- **Contexto del estado actual**  
- **Archivos a tocar (rutas exactas)**  
- **Pasos: abrir → pegar → guardar**  
- **Criterio de done (verificable)**  
- **Checklist UI/SEO/OG**
